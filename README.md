# Niloy Rahman Portfolio

Personal portfolio for Niloy Rahman, focused on AI/ML engineering, retrieval systems, applied research, and production-minded ML projects.

Live site: https://niloy37.github.io/portfolio

## What It Includes

- A single-page Next.js portfolio with sections for profile, experience, projects, skills, and contact.
- Featured case studies for RemoteTrust AI, RAG, resume parsing, DDoS detection, and PadChest X-ray classification.
- Project proof through GitHub repositories, live demos, and LinkedIn walkthrough embeds.
- Static export configuration for GitHub Pages.
- Contact form integration through Formspree.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons

## Local Development

```bash
npm install
npm run dev
```

The local app runs at `http://localhost:3000`.

## Build

```bash
npm run build
```

The project is configured with `output: 'export'` in `next.config.js` for static hosting.

## Content Map

- `data/personal.ts` - profile, experience, education, awards, and social links
- `data/projects.ts` - featured and archive project case studies
- `data/skills.ts` - skill clusters, toolbelt, and working style
- `components/` - page sections and interactive UI
- `public/assets/resume.pdf` - downloadable resume
- `public/favicon.svg` and `public/favicon.ico` - site favicon assets
- `public/og-image.svg` - social sharing preview

## Deployment

The site is intended for GitHub Pages at `https://niloy37.github.io/portfolio`. The production `basePath` and `assetPrefix` are set to `/portfolio` in `next.config.js`.

For a manual static build, run:

```bash
npm run build
```

Then deploy the exported output generated by Next.js.

Project screenshots are not currently part of the live UI. The portfolio uses live links, GitHub repositories, and LinkedIn embeds as proof links for the selected work.
